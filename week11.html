WEEK 11

DAY 1

Since the custom events for Iran travel tours have already been created, I published each one in Google Tag Manager to activate the changes I make whenever I modify parameters. These custom events include view item, view cart, thank you page, and purchase.

DAY 2

The next day, I continued what I was doing yesterday, which was publishing the custom events I had created before. Now, the custom events I have published include removed item, menu link click, log in, and inquiry form, configuration, and begin check out.

DAY 3

The following day, I proceeded to publish additional custom events, which include add payment info, FB - view item, FB - view cart, FB - remove item, FB - purchase, and FB - page view.

DAY 4

The next custom events I published were FB - add payment info, FB - add to cart, FB - begin checkout, and FB - inquiry form.

DAY 5

The next day, I created a tag for the contact us page and added a parameter. After creating it, I checked if it was capturing the necessary data for my GA4 using Google Tag Assistant and Debug View. Once confirmed, I published the changes live.
